Tutorial Sitio Web con AWS RDS - EC2 - S3

Escuela Colombiana de Ingeniería Julio Garavito · Amalia Alfonso

En este turorial se explicará cómo realizar un sitio web dinámico haciendo uso de los servicios de AWS. El sitio hará uso de 3 servicios de AWS: EC2, S3 y RDS.
En S3 se desplegar'an archivos estáticos, que usará la app en java de EC2.
En EC2 se desplegará una aplicación de java con REST usando Spring Boot, la cual responderá a peticiones desde S3.
En RDS se creará una base de datos que permitará almacenar y cargar datos al sitio web.

El repositorio está disponible en: WebApp-AWS-BD WebApp-AWS-BD
